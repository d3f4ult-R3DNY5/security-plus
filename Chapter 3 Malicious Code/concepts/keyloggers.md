## Keyloggers 

### Overview 
---
>[!info]
>The keyloggers are programs that can capture the **keystrokes** from input devices such as 
>- mouse movement
>- touchscreen inputs
>- credit card swipes
>- keyboards 

### Related IOC 
---
>[!example] Indicators of Compromise
>- File hashes and Signaturees
>- Exfiltration activity to command and control systems
>- Process names
>- Known URL's
